# Cloud Storage Platform (Lifetime Free Storage) BY PRIYANSHU 

A full-stack cloud storage platform that allows users to upload, store, and manage files such as photos, videos, notes, and PDFs securely for lifetime access.
The system is built using advanced backend architecture and database management to ensure fast, scalable, and secure storage handling.

## Features
- User authentication (Login/Signup)
- Upload images, videos, PDFs, notes
- File preview & download
- Organized file management (folders/categories)
- Secure cloud storage system
- Lifetime file access
- Fast upload & retrieval system
